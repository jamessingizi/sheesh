# Swoosh

A collection of useful IT tools in the commandline

### Utilities

- slugify
- uuid
- ulid
- base64 converter file
- base64 converter text
- basic auth generator
- bcrypt
- case converter
- chmod calculator


## Development
activate poetry env with
```shell
poetry shell
```

### Testing
Running tests
```shell
poe test
```

Linting
```shell
poe lint
```
